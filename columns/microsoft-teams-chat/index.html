電子黒板×Microsoft Teamsでチャット参加型の授業をつくる方法

公開日: 2026年6月5日

読了 約7分

授業中に「意見がある人？」と聞いても、手が挙がるのは一部の生徒だけ。そんな場面は少なくありません。意見は持っていても、人前で話すのが苦手な生徒もいます。発表者に直接コメントするのは緊張する生徒もいます。時間の都合で、全員の意見を聞くことが難しい場合もあります。

そこで活用したいのが、電子黒板とMicrosoft Teamsのチャット機能です。電子黒板にTeamsのチャット画面を表示し、生徒は手元のタブレットからコメントを投稿します。発表を聞きながら、気づいたこと、質問、良かった点、改善点をリアルタイムで書き込む。これにより、手を挙げて発言する形式よりも、多くの意見を短時間で集められます。

これまでの授業で起こりがちな課題

従来の授業では、意見を集める方法として「手を挙げて発言する」スタイルが一般的でした。もちろん、口頭で発言する力は大切です。しかし、この方法だけでは拾える意見に限りがあります。発言できる人数は時間に左右されますし、発言する生徒も固定化しがちです。特に中学生以上の学校環境では、多くの生徒がタブレット端末を持っています。それなら、発言の方法を「声」だけに限定する必要はありません。

チャット機能を使えば、発表を聞く側の生徒全員が、同時にコメントを投稿できます。1人ずつ当てていく方法では数人の意見しか聞けなくても、チャットなら短時間で多くの意見を集められます。場面によっては、手を挙げる形式の何十倍ものコメントが集まることもあります。大切なのは、声の大きい生徒だけで授業を進めないことです。チャットを使えば、普段は発言しない生徒の考えも、教室全体に共有できます。

活用例①：発言しにくい生徒の意見を拾う

クラスには、手を挙げて発言するのが得意な生徒もいれば、苦手な生徒もいます。特に、全員の前で話すことに抵抗がある生徒は、よい意見を持っていても発言できないことがあります。授業後にノートを見ると、実は深く考えていたことが分かる。そんな経験をした先生も多いのではないでしょうか。Teamsチャットを使うと、そうした生徒の意見を拾いやすくなります。

※画像はイメージです

口頭で話す必要はなく、タブレットから短く投稿できます。文章であれば、自分の考えを落ち着いて書ける生徒もいます。電子黒板にチャットを表示すれば、投稿された意見を教室全体で共有できます。先生はその中から気になるコメントを取り上げ、「この意見は面白いですね」「この視点をもう少し考えてみましょう」と授業に活かせます。普段はあまり話さない生徒が、チャットでは一番活発に意見を出すこともあります。隠れていた思考や才能が見つかるきっかけにもなります。

活用例②：時間制約に縛られず、多くの意見を集める

授業中に意見を集めるとき、もっとも大きな制約になるのが時間です。「この発表について、意見がある人？」と聞いて、手を挙げた生徒を一人ずつ指名する。これも大切なやり取りですが、授業時間の中で発言できる人数には限りがあります。1人が話している間、他の生徒は待つことになりますし、発言する生徒も固定されがちです。

一方、Teamsチャットを使えば、全員が同じタイミングでコメントできます。これにより、短い時間でも多くの意見を集められます。

たとえば、発表後に3分間だけコメント時間を設けるとします。手を挙げる形式であれば、発言できるのは数人かもしれません。しかしチャットであれば、クラス全員が同時に投稿できます。30人のクラスなら、1人1コメントでも30件の意見が集まります。生徒によっては、良かった点と改善点を分けて2件、3件と投稿することもできます。発表者にとっても、1人か2人の感想ではなく、クラス全体から反応を受け取れる点が大きなメリットです。「声が聞き取りやすかった」「スライドの文字が少し小さかった」「最初に結論を言うともっと分かりやすい」「具体例がよかった」など、複数の視点から改善点を確認できます。

また、先生にとっても、チャットは授業を調整するための材料になります。リアルな情報を見れば、「次の発表ではここを意識しましょう」「今出ている質問にまとめて答えます」と、その場で授業に反映できます。チャットを使うことで、意見収集は「一人ずつ発言を待つ時間」から、「全員が同時に参加する時間」に変わります。時間制約に縛られず、より多くの声を授業に取り入れられるのです。

活用例③：保護者会で質問・感想をリアルタイムに集める

Microsoft Teamsのチャット機能は授業内の意見共有に向いていますが、保護者会で保護者様をTeamsに招待し、チャットに参加してもらう運用は少し手間がかかります。アカウントの有無やログインの問題があり、当日の案内も複雑になりがちです。そこで保護者会では、Googleフォームを使った質問受付がおすすめです。

事前に「質問・感想フォーム」を作成し、QRコードを生成します。当日は、そのQRコードを含んだスライドを電子黒板に大きく表示します。保護者様はスマートフォンでQRコードを読み取り質問や感想を投稿できます。先生側は、Googleフォームの回答をスプレッドシートで確認します。投稿内容をリアルタイムで確認できるため、似た質問をまとめて回答したり、時間内に扱えなかった質問を後日フォローしたりできます。

この方法なら、保護者様にTeamsへ参加してもらう必要がありません。スマートフォンでQRコードを読み取るだけなので、参加のハードルが低くなります。また、回答がスプレッドシートに蓄積される点も大きなメリットです。保護者様がどのような点に不安を感じているのか、どの説明に質問が多かったのかを後から振り返れます。次回の保護者会や個別面談、配布資料の改善にも活かせます。保護者会は、一方的に説明するだけではなく、保護者様のリアルな声を集める場でもあります。電子黒板にGoogleフォームのQRコードを表示すれば、質問しやすく、学校・塾側もデータとして活用しやすい運用ができます。

運用のポイント：投稿ルールを最初に決める

Teamsチャットを授業で使う場合は、最初に簡単な投稿ルールを決めておくことが大切です。たとえば、発表者を傷つける言葉は使わない。良かった点と改善点を分けて書く。質問は短く具体的に書く。関係のない投稿はしない。ルールを厳しくしすぎる必要はありません。ただ、安心して投稿できる場にするためには、最低限の約束が必要です。

おすすめは、コメントの型を用意することです。

「よかった点：」
「質問：」
「もっとよくなる点：」
「印象に残った言葉：」

このような型があると、生徒はコメントしやすくなります。先生も投稿を整理しやすくなります。

まとめ：チャット機能で、授業参加の入口を増やす

電子黒板とMicrosoft Teamsを組み合わせると、授業参加の形が広がります。

手を挙げなくても意見を出せる。
発表中にリアルタイムでコメントできる。
普段発言しにくい生徒の考えを拾える。
プレゼンの改善点をその場で共有できる。
保護者会でも質問をタイムリーに集められる。

授業は、一方通行の講義だけではありません。生徒の「分かった」「気づいた」「質問したい」を拾い上げ、双方向に作り上げていくものです。

Microsoft Teamsのチャット機能を電子黒板に表示すれば、その場の思考が可視化されます。教室の中で声にならなかった意見も、チャットを通じて共有できます。

「手を挙げる」勇気がなくても、授業に参加できる。

それが、電子黒板×Teamsチャット活用の大きな価値です。
