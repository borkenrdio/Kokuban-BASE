## 2. SEO面の対応

### 2-1. 旧URLの行き止まり解消【改善レポートH-3】
放置されていた旧ページに、新ページへの転送（0秒meta refresh + canonical + noindex）を設置。

| 旧URL | 転送先 |
|---|---|
| /Kokubanlease | /lease/ |
| /reservation | /experience/ |
| /on-line-reservation | /experience/ |
| /whatissmrtboard | /denshikokuban/ |
| /terms | /company/ |
| /online-consultation | /support/（旧guide経由の二段転送を直行化） |
| /guide | /support/（ページ統合に伴い新設） |

※ GitHub Pagesの制約上、HTTP 301ではなくmeta refresh方式（既存 /column/ と同方式）。Googleは恒久移転として扱います。Search Consoleでの「インデックス登録リクエスト」推奨。

### 2-2. トップページの見出し復元【H-4】

## 5. 新機能

### 5-1. 30秒で分かる 電子黒板選びチェック（/check/）
- **5問**（授業形態 / 使い方 / 教材環境 / 重視点2つ / 予算観）のポップな診断UI。1問ずつカード表示・自動進行・進捗バー・戻る対応
- **結果表示**: 製品写真 + メーカーロゴ + インチ数 + **リース目安月額**（65型=掲載価格、**75型以上=+3,000円**、27/55型=お見積もり+参考価格）+ 選んだ理由（回答連動で最大3件）+ CTA3種 + 次点候補
- **公平性**: 全810通りの回答組合せをシミュレーションし、1位獲得率を BenQ 23.4% / StarBoard 21.5% / MIRAI 19.9% / Promethean 19.2% / SHARP 15.9% に調整。全ブランドに明確な勝ち筋があり、同点時は2ブランドを対等に並列表示
- ブランド出し分けは指定方針どおり（個別・少人数GW→MIRAI 27型最有力 / 中心利用→StarBoard / ICT・AI→BenQ / 費用重視→SHARP / 書き心地・使いやすさ→Promethean）

### 5-2. 診断への導線
- トップ・ラインナップに**グラデーションバナー**、コラム一覧サイドにミニバナー
- **フローティング誘導**（全ページ共通・site.js注入）: 入場直後には出さず、①一定スクロール後 または ②セッション3ページ目で表示。**×で7日間非表示**。/check/ と /contact/ では非表示。右端吸着のコンパクトデザイン

### 5-3. コラム一覧の全面リニューアル（/article/）
- **カテゴリ絞り込みチップ**（件数付き・スクロール追従・タップで即絞り込み）— microCMSの新5カテゴリ対応（旧カテゴリ名は自動変換）
- **高密度グリッド**（PC3列/モバイル2列）+ タイトル帯を1行に圧縮 → ファーストビューから記事が見える
- **編集部Pick Up**サイドバー（独自取材の最新5件を自動選定・スクロール追従）
- カードに**説明文2行**を追加、**著者表示は廃止**、GA計測タグは維持、Tailwind CDN依存を撤廃
- ビルドスクリプト（fetch_and_build.mjs）とテンプレートを更新済みのため、**今後microCMSで記事公開すれば自動的にこのデザインで再生成**

## 6. 障害対応

- **index.htmlの文字化け破損を検知・復元**（7/5）: 外部エディタ保存時のエンコード事故（UTF-8→Shift-JIS二重変換）で全日本語が破損。直前コミットに正常版が残っていたため復元。→ **今後index.html等を編集する際はUTF-8保存を徹底してください**
