## 1-3. 課題の詳細（優先度つき）

### 【優先度：高】

**H-1. 「電子黒板 価格」に応えるページがない**
> **【2026-07-04 決定事項】** 価格はまずコラム記事として執筆する方針。サイト本体(製品・比較・リースページ)への金額掲載は、メーカーとの調整を進めた上で改めて判断する。当面、本項の「価格ページ新設」は実装しない。
- **どの検索意図か**: 「電子黒板 価格」「電子黒板 値段 相場」「電子黒板 65インチ 価格」。導入を具体的に考え始めた、問い合わせに最も近いユーザー。
- **なぜ今のサイトでは足りないか**: 「電子黒板とは」に「本体だけでなく設置・保証まで見て考えよう」という考え方の説明はあるが、サイト全体で具体的な金額が1円も出てこない（リースの月額例すら、過去のプレスリリース記事に12,603円とあるだけ）。価格を検索した人は相場の数字を求めており、数字がないページは検索でも選ばれず、訪問しても離脱する。
- **問い合わせへのつながり**: 「サイズ別の本体相場＋設置費・保証の目安＋購入とリースの月額比較」を示した上で「正確な金額は構成で変わるので見積もり相談へ」と導くのが王道の導線。価格ページは集客と問い合わせ獲得を同時に担う、優先度最高のページ。

**H-2. 「補助金・助成金」コンテンツが皆無**
- **どの検索意図か**: 「電子黒板 補助金」「ICT 導入 補助金 私立」「学習塾 IT導入補助金」。予算の裏付けを探している=導入意思が固いユーザー。
- **なぜ足りないか**: 学校・塾の設備投資は補助金と切り離せないのに、サイト内にまとまった情報がない。競合の販売店サイトはこのテーマで集客している。
- **問い合わせへのつながり**: 「使える制度の整理＋申請スケジュール＋対象機種の相談」という形で、そのまま相談予約に接続できる。制度情報は更新が必要なので、更新日を明示して運用する前提で作る。

**H-3. 旧ページのURLが行き止まりになっている**
- 旧「電子黒板とは」ページ（whatissmrtboard）、旧リース紹介ページ（Kokubanlease）、来場予約・オンライン予約ページが削除され、跡地が何も表示されない状態。旧リースページはプレスリリースや外部サイトからリンクされていた可能性が高い。
- **影響**: これまで検索エンジンに蓄積された評価と、外部リンクからの流入がそのまま捨てられている。新ページ（電子黒板とは / リース）へ案内する引っ越し先の案内を用意すべき（コラム一覧の旧URLでは既に同様の案内ページが用意できているので、同じやり方でよい）。
- **工数が小さいのに損失が続いているため、即対応推奨。**

**H-4. トップページの大見出しがブランド名のみ**
- 現在のトップページの最重要見出しは「Kokuban BASE」だけで、「電子黒板」という言葉が入っていない（「電子黒板選びの総合窓口」は小さな添え書き扱い)。検索エンジンはページの最重要見出しを重視するため、社名しか書いていないのは機会損失。以前のSEO改善で「見出しにキーワードを入れる」対応をした経緯があるが、その後のデザイン刷新で後退している。

### 【優先度：中】

**M-1. 「電子黒板 比較」で勝てる比較の中身になっていない**
- ラインナップページは構成（比較表＋選び方3基準＋ブランドカード）は良いが、機能比較表がほぼ全項目「〇」。検索上位の比較コンテンツは「違いがわかる」ことが価値なので、このままでは上位を狙いにくい。価格帯・保証年数・サイズ展開・得意な授業スタイルなど、実際に差がつく軸を加える必要がある。

**M-2. 「学校向け」「学習塾向け」の専用受け皿がない**
- 「電子黒板 学校」「電子黒板 学習塾」は主要ターゲットの検索なのに、それぞれに応える専用ページがない（トップに案内カードはあるが、リンク先は共通の「できること」）。学校は稟議・年度予算・全校導入、塾は塾長の即断・リース・1教室からと、意思決定がまったく違う。それぞれの言葉で書かれたページがあれば、検索の受け皿と問い合わせ前の「自分向けだ」という納得の両方が得られる。

**M-3. ショールーム・実機体験の専用ページがない**
- 「実機を触って比べられる」ことはこのサイト最大の差別化点なのに、「できること」ページ内のセクションに埋まっており、「電子黒板 ショールーム」「電子黒板 体験 京都/大阪」の検索に応えるページがない。専用ページ化し、あわせてGoogleビジネスプロフィール（地図検索対策）への登録を進めるべき。

**M-4. 49本の記事が新着順に並んでいるだけで、テーマ別の入口がない**
- コラム一覧は「新着・独占取材・コラム」の区分のみ。「選び方の記事だけ読みたい」「国語の活用例を探したい」ができない。テーマ別（選び方 / 教科別活用 / メーカー取材）のまとめページを作ると、ロングテール検索の受け皿が増え、サイト内回遊も伸びる。

**M-5. ご利用ガイド・できること・導入サポートの役割重複**
- 3ページとも「相談から導入までの流れ」「何を支援するか」を語っており、検索エンジンから見るとテーマが分散している。「相談窓口」「導入サポート」系の検索評価が3ページに割れるより、役割を明確に分けるか統合した方が強い。

### 【優先度：低】

- **L-1. FAQが6問と少ない**: 価格・納期・設置工事・保証・故障対応・他社購入品の相談可否など、営業現場で実際に聞かれる質問を追加していくと、検索の受け皿とリッチリザルト(検索結果での目立つ表示)の両方に効く。
- **L-2. 「電子黒板とは」ページのFAQに構造化マークの未設定**: 検索結果で質問が直接表示されるチャンスを取りこぼしている。
- **L-3. 京都・関西ローカル検索の深掘り**: 「電子黒板 京都」等は以前の対策で着手済み。ショールームページ（M-3）とセットで進めるのが効率的。

## 1-4. 集客記事と問い合わせページの役割分担

骨格はできているが、**「集客記事」と「問い合わせ」の間をつなぐ中間ページが不足**している。現状は「記事を読む → いきなり無料相談」の二択に近い。価格ページ（H-1）、補助金ページ（H-2）、学校向け・塾向けページ（M-2）、体験ページ（M-3)は、いずれも「記事より問い合わせに近いが、フォームより心理的に軽い」中間の受け皿であり、ここを埋めることが問い合わせ数の増加に直結する。

# 第2部 サイト訪問後のデザイン・導線の現状分析

対象ユーザー: 電子黒板の導入を初めて検討する学校・学習塾の担当者（専門知識なし）。

## 2-1. できていること（強み）

- トップページ直下の**目的別メニュー（電子黒板とは / 比較・選び方 / リース・費用 / 相談）**は、初心者が最初の一歩を選べる良い設計。
- 「学校の先生へ」「学習塾の運営者へ」と**相手別に語りかけるカード**があり、自分ごと化しやすい。
- 下層ページに**画面下部の固定ボタン（電話＋問い合わせ）**があり、どこまで読んでも相談に進める。
- 比較ページのCTAが「相談する / 実機体験する / リースを見る」と**次の行動を3択で示している**のは優秀。
- 製品ページの「Kokuban BASEスタッフより」コメントは、単なるカタログ転載ではない価値がある。

## 2-2. 課題の詳細

### トップページ：第一印象と「何の会社か」

1. **最初に目に入る最大の文字が「Kokuban BASE」という社名**で、「電子黒板選びの総合窓口」が小さい。初訪問者(特に検索から来た人)は0.5秒で「自分に関係あるページか」を判断するため、**「何をしてくれるか」を最も大きく、社名は従に**すべき。
2. **「標準搭載機能について」の説明画像が用意されておらず、表示が欠けた状態**になっている（画像ファイル未設置)。トップページの画像切れは信頼感を直接損なうため、即修正が必要。ヒーロー画像等にも「差し替え予定」のまま残っている箇所があり、公開前チェックの仕組みが必要。
3. **「最新NEWS」欄に、お知らせではなくコラム記事が表示される**仕組みになっている。「NEWS」というラベルと中身が一致しておらず、お知らせページ(別に存在する)とも二重管理状態。
4. **トップページに5ブランドの「顔見せ」がない。** 何を扱っている店なのかは、ブランドロゴや製品写真が並んでいて初めて直感的に伝わる。現状、製品に触れるのは目的別メニューの「比較・選び方」の文字リンクだけで、「品揃え」が見えない。ブランドロゴ帯＋ラインナップへの誘導を1セクション入れるだけで印象が変わる。
5. 下層ページにはある**画面下部の固定相談ボタンが、トップページにだけない。**

### 初心者の迷い：3つの似たページ

6. **「ご利用ガイド」「Kokuban BASEでできること」「導入サポート」の違いが初見でわからない。** ナビに8項目が並び、うち3つが「サービス説明」。どれを読めばいいか迷った人は、どれも読まない。
   - **整理の方向性**: 「できること」（サービス全体・ショールーム・デモ機）を軸に1本化し、「ご利用ガイド」は流れの説明として「できること」内へ、「導入サポート」の選ぶ・買う・使うの内容も統合する——など、**「サービスを知るページは1つ」**にするのが理想。少なくともナビの並びと名前で違いがわかるようにする。

### 「電子黒板とは」ページ：長さと案内

7. 内容は充実しているが、**11章の長いページなのに冒頭に目次がない。** 「価格だけ知りたい」「サイズだけ知りたい」人が目的の場所に飛べない。冒頭に章立ての目次（とくに「サイズ」「価格」「選び方チェック」への近道）を置くべき。コラム記事には目次の自動表示があるのに、このページにはない。

### 比較・製品ページ：選んだ後の「いくら？」が答えられない

8. **機能比較表がほぼ全部「〇」で、比較しても違いがわからない**（第1部M-1と同根）。初心者が知りたいのは「どれも同じくらい高機能」ではなく「うちの場合はどれか」。価格帯・保証・サイズ展開・得意分野など差が出る行を追加し、「あなたの現場ならこれ」に落ちる表にする。
9. **製品ページを読み終えても価格の目安がどこにもない。** 検討者は比較の次に必ず「いくらか」を考える。目安すらないと、問い合わせのハードルだけが残る。「価格は構成による」なら、せめて「〇〇型クラスの導入例: 月額〇円〜」のような幅のある目安を。
10. **製品ページから問い合わせるとき、機種の情報が引き継がれない。** 問い合わせフォームの種別選択に「どの機種を検討中か」がなく、自由記入に頼っている。「この機種について相談する」と押した人が、フォームでまた説明し直す形になっている。

### 問い合わせ導線：ハードルが一段高い

11. **フォームは全項目必須で、部署・役職や電話番号まで入力しないと送信できない。** 検討初期の担当者(特に情報収集段階)には重い。必須は「種別・名前・連絡先1つ」程度に絞り、他は任意にするのが一般的。
12. **「フォームか電話か」の二択しかなく、軽い相談の受け皿がない。** FAQは6問のみ、資料請求・オンライン相談予約のような「まだ営業されたくない段階」向けの選択肢がない。BtoBでは「まず資料だけ」の需要が大きい。

### 信頼感・安心感

13. **体験談がすべて匿名**（方針として明記されているのは誠実だが)、学校・塾の実名事例が1件もないため、「本当に導入実績があるのか」を証明できていない。実名・写真つき事例が1〜2件あるだけで説得力が大きく変わる。
14. **Kokuban BASE自身の実績数字がない。** メーカーの「50万台」「7万台」は載っているが、「相談〇件」「導入支援〇教室」「运営会社は京都で学習塾を運営」といった自社の裏付けがトップに出ていない。運営チームページ(現役講師が案内)は良い材料なのに、トップから見えない。

### モバイル

15. 直近でモバイル改善が実施済みであり、表の横スクロール対応などの基本はできている。残る確認ポイントは3つ：
    - 比較表(5ブランド×12機能)がスマホで「横にスクロールできる」と気づける見た目になっているか
    - 画面下部の固定ボタン(電話＋問い合わせ＋営業時間表示)が画面を占有しすぎて本文やフッターに被っていないか
    - トップの目的別メニュー4枚がスマホで縦に長くなりすぎていないか

## 実装前に方向性を確認したい重要な判断ポイント

1. **価格をどこまで公開するか。** → **決定済み(2026-07-04)**: まずコラム記事として執筆。サイト本体への金額掲載はメーカーとの調整を進めた上で改めて判断。今後の論点は「コラムにどこまで具体的な数字を載せるか」と「調整がついた後、どのページから金額掲載を始めるか」の2点。
2. **サービス説明3ページ(ガイド/できること/サポート)を統合するか、残して役割を分けるか。** サイト構成の根幹に関わるため、先に方針を決めたい。
3. **実名事例を出せる顧客がいるか。** 掲載許諾が取れる導入先が1件でもあれば、信頼面の打ち手が大きく変わる。
4. **主戦場を学校と学習塾のどちらに置くか。** 両方対応は維持しつつ、ページの筆頭・広告的な訴求順をどちらに寄せるかで、トップページと新設ページの設計が変わる。
